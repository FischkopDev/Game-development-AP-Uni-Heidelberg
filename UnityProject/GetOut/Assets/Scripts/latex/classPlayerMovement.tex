\doxysubsection{Detailed Description}
Handles the movement logic for the player character in Unity. 

This script allows the player to move in various directions using keyboard input, and it manages both walking and running speeds. The script is designed to work with Unity\textquotesingle{}s Rigidbody component for physics-\/based movement.

Features\+:
\begin{DoxyItemize}
\item Supports basic movement (forward, backward, left, right) using WASD or arrow keys.
\item Allows the player to sprint by holding down a designated key (e.\+g., Left Shift).
\item Implements smooth movement by interpolating the player\textquotesingle{}s velocity.
\item Adjusts movement speed based on whether the player is walking or running.
\item Includes optional jump functionality if the player is grounded.
\end{DoxyItemize}

Requirements\+:
\begin{DoxyItemize}
\item This script should be attached to a Game\+Object with a Rigidbody component.
\item The Game\+Object should also have a Collider component for proper collision detection.
\end{DoxyItemize}

How to Use\+:
\begin{DoxyItemize}
\item Attach the \mbox{\hyperlink{classPlayerMovement}{Player\+Movement}} script to your player Game\+Object.
\item Configure movement speed, running speed, and jump force in the Unity Inspector.
\item Ensure the Rigidbody component is set to \char`\"{}\+Interpolate\char`\"{} for smooth movement.
\item Set the correct input axes in the Unity Input Manager (default is WASD or arrow keys).
\end{DoxyItemize}
