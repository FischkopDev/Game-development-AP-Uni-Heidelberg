\doxysubsection{Detailed Description}
Handles the interaction logic for readable items. 

This component is designed to be attached to gameobject. Providing an id you\textquotesingle{}ll be able to read items.

Features\+:
\begin{DoxyItemize}
\item Show text when clicking on an object containing this component
\end{DoxyItemize}

Requirements\+:
\begin{DoxyItemize}
\item Check if the item is clicked with \char`\"{}\+E\char`\"{}
\item 
\end{DoxyItemize}

How to Use\+: Give this component to any gameobject you\textquotesingle{}d like to cleanup or make disappear when clicked.
