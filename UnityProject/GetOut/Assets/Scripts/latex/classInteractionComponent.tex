\doxysubsection{Detailed Description}
Handles the interact-\/action between player and gameobjects. 

This component checks if the player is performing an interaction at the moment

Features\+:
\begin{DoxyItemize}
\item Trigger for interaction
\item Receive the hit object
\item Check if the object is in a specific range for interaction
\end{DoxyItemize}

Requirements\+:
\begin{DoxyItemize}
\item Check each frame for interaction
\end{DoxyItemize}

How to Use\+: Give this component to the player in unity and add another class using @\+Interactable to any gameobject you\textquotesingle{}d like to interact with.
