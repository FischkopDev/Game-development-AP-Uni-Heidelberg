\doxysubsection{Detailed Description}
Handles the interaction logic for cleaning up or disabling objects in the game. 

This component is designed to be attached to objects that should be disabled or \char`\"{}cleaned up\char`\"{} when the player interacts with them. It implements the \mbox{\hyperlink{interfaceInteractable}{Interactable}} interface.

Features\+:
\begin{DoxyItemize}
\item Disable gameobjects when clicked
\end{DoxyItemize}

Requirements\+:
\begin{DoxyItemize}
\item Check each frame for interaction
\item Make gameobjects invisible that got cleaned up
\end{DoxyItemize}

How to Use\+: Give this component to any gameobject you\textquotesingle{}d like to cleanup or make disappear when clicked.
